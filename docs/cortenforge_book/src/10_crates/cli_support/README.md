# cli_support

## Overview
Shared CLI args and helpers for tools and apps, with optional Bevy resource integration.

## Usage
Import helper types in tools/apps; docs.rs: https://docs.rs/cortenforge-cli-support; source: https://github.com/via-balaena/CortenForge/tree/main/crates/cli_support.

## Pitfalls
Keep defaults light to avoid pulling heavy deps in simple CLIs.
