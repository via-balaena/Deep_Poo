# data_contracts

## Overview
Schemas and validation helpers for capture metadata and warehouse manifests.

## Usage
Use in capture_utils/burn_dataset/training to validate runs; docs.rs: https://docs.rs/cortenforge-data-contracts; source: https://github.com/via-balaena/CortenForge/tree/main/crates/data_contracts.
