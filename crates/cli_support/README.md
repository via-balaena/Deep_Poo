# cli_support

Shared CLI argument helpers for CortenForge tools and apps: capture/warehouse output options, thresholds/weights, seed resolution, and optional WGPU hints.

## Features
- No default features.
- `bevy-resource`: derives `Resource` for `SeedState` by enabling a lightweight Bevy import (default-features off).
